## 2. Core Features You Must Master

To truly master Cursor, you need to understand the three pillars of its AI interaction: **Chat (Cmd+K/L)**, **Tab (Predictive Editing)**, and the revolutionary **Composer (Cmd+I)**.

### A. AI Chat (Cmd+L)
This is your persistent sidekick. Unlike a browser-based LLM, Cursor's Chat allows you to:
- **@ Files**: Instant context injection by mentioning specific files.
- **@ Codebase**: Trigger a search across your entire project.
- **@ Folders**: Narrow the AI's focus to a specific module.

*Pro Tip: Use Cmd+L to ask "How does this function connect to the database?" and let it trace the logic across five different files.*

### B. Inline Fixes (Cmd+K)
Highlight a block of code, hit `Cmd+K`, and tell the AI what to change. 
- *Input:* "Convert this from Javascript to Typescript."
- *Result:* Precision editing without copy-pasting code back and forth from a chat window.

### C. The Game Changer: Composer (Cmd+I)
If Chat is the sidekick, Composer is the Architect. In Composer mode, the AI can **create and modify multiple files simultaneously**. You can literally give a high-level instruction like:
> "Create a new contact page with a form that sends data to our existing API endpoint, and add a link to it in the footer."

Composer will write the new page, update the footer component, and ensure types are consistent—all in one go.
